## A Vanilla Web Prototyping boilerplate

![Wolfie!](https://github.com/juanmnl/proto/blob/master/src/public/assets/wolfang.svg)

* Nodemon
* Webpack
* Babel
* PostCSS
  * CSSnext
* Faker.js (design with data)

**\*How to use it?**

`git clone`  
`npm install`  
`npm start`  
✅
